防抖and节流

2021-03-25 性能优化 JavaScript

防抖节流

性能优化

防抖

在第一次执行函数的时候，给定一个值，如果在这个时间内没有再次触发，就执行，否则当前计时取消，重新开始计时

任务频繁触发的情况下，只有任务触发间隔超过了指定的间隔的时候，任务才会执行。

//防抖---某个时间段内只处理一次
    function debounce(fn,delay){
      let timer = null;
      return function(){
        //每次点击的时候将前一个定时器清除
        clearTimeout(timer);
        timer = setTimeout(()=>{
          fn.call(this,arguments)
        },delay)
      }
    }

节流

指定的时间内只会执行一次

  • 懒加载监听滚动条的位置，使用节流函数按照一定的时间频率获取
  • 用户点击提交，使用节流，只允许一定时间内执行一次
  • TIPS：为什么需要节流？
    • 一些操作可能会造成浏览器回流，造成过多的损耗，使用节流来防止这种情况
function throttle(fn,delay){
      let canRun = true;//闭包
      return function(){
        if(!canRun)return 
        //防止执行之前 再被执行
        canRun = false
        setTimeout(()=>{
          fn.call(this,arguments);
          canRun = true
        },delay)
      }
    }

节流还可以利用时间戳，差值是否大于指定间隔
