JavaScript不借助第三个变量交换两个变量的值

2021-04-20
let a = 1,b = 2;
  • 借助了es6的中结构赋值方法
[a,b] = [b,a]
  • 先把a作为一个对象，赋值键值对，取出键值对
a = {a:b,b:a}
a = a.a;
b = a.b;
  • 数组：类似上一种方法
a = [a,b]
a = a[1]
b = a[0]
  • 异或
a ^= b;
b ^= a;
a ^= b;
  • 加减
a = a + b;
b = a - b;
a = a - b;
